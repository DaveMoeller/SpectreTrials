# SpectreTrials
**Designed and built by MJM Games**
Spectre Trials
Basic maze game with levels of difficulty.
## Keys
The following are navigation keys:

- Left and right
	* a/d Keys
	* Left, Right Arrow Keys
	* Left/Right Mouse Buttons
- Up and Down
	* w/s Keys
	* Up and down arrow keys
	* Forward and back mouse keys
